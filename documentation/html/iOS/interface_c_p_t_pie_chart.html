- (id) initWithFrame: (CGRect)  newFrame
implementation

Initializes a newly allocated CPTPieChart object with the provided frame rectangle.

This is the designated initializer. The initialized layer will have the following properties:

  • pieRadius = 40% of the minimum of the width and height of the frame rectangle
  • pieInnerRadius = 0.0
  • startAngle = /2
  • endAngle = NaN
  • sliceDirection = CPTPieDirectionClockwise
  • centerAnchor = (0.5, 0.5)
  • borderLineStyle = nil
  • overlayFill = nil
  • labelRotationRelativeToRadius = NO
  • labelOffset = 10.0
  • labelField = CPTPieChartFieldSliceWidth
Parameters
newFrameThe frame rectangle.
Returns
The initialized CPTPieChart object.

- (BOOL) pointingDeviceDownEvent: (CPTNativeEvent *)  event
atPoint: (CGPoint)  interactionPoint 
implementation

Informs the receiver that the user has touched the screen.

If this plot has a delegate that responds to the -pieChart:sliceWasSelectedAtRecordIndex: and/or -pieChart:sliceWasSelectedAtRecordIndex:withEvent: methods, the interactionPoint is compared with each slice in index order. The delegate method will be called and this method returns YES for the first index where the interactionPoint is inside a pie slice. This method returns NO if the interactionPoint is outside all of the slices.

Parameters
eventThe OS event.
interactionPointThe coordinates of the interaction.
Returns
Whether the event was handled or not.
