Detailed Description

Immutable wrapper for various text style properties.

If you need to customize properties, you should create a CPTMutableTextStyle.

Method Documentation

- (id) init
implementation

Initializes a newly allocated CPTAnnotation object.

The initialized object will have the following properties:

  • fontName = Helvetica
  • fontSize = 12.0
  • color = opaque black
  • textAlignment = CPTTextAlignmentLeft
