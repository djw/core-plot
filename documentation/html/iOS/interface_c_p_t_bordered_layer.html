Detailed Description

A layer with a border line and background fill.

Sublayers will be positioned and masked so that the border line remains visible.

Method Documentation

- (id) initWithFrame: (CGRect)  newFrame
implementation

Initializes a newly allocated CPTBorderedLayer object with the provided frame rectangle.

This is the designated initializer. The initialized layer will have the following properties:

  • borderLineStyle = nil
  • fill = nil
  • inLayout = NO
  • needsDisplayOnBoundsChange = YES

newFrameThe frame rectangle.
